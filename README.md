# apertium-fin-krl

Apertium translator between Finnish and Karelian (North / Vienankarjala /
"proper" or so)

## Requirements

You will need the following software installed:

* lttoolbox
* apertium
* apertium-fin
* apertium-krl

If this does not make any sense, we recommend you look at:
http://wiki.apertium.org
